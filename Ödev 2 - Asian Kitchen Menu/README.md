## Ödev 2 Asian Kitchen Menu

Hazır verilen HTML, CSS dosyalarına ek isterlere göre Javascript dosyası yazıldı.

1) Sayfayı açınca tüm menü öğelerini gösteriyor. 
2) All, Korea, Japan, China butonlarını oluşturuyor. 
3) Butona tıklayınca o kategorideki yemekleri filtreliyor.
